### 流式操作是什么

![](assets/004/01/01/01-1579076589455.png)

```
java 8新特性
流式操作：不是一个数据结构，不负责任何的数据存储。
更像是一个迭代器。有序的获取到数据源中的每一个数据，并且可以对这些数据进行一些操作。
流式操作的每一个方法，返回值都是返回的流本身。

```

* 三个步骤

1. 获取数据源：集合、数组
2. 对数据进行处理的过程：过滤、排序、映射。。。（中间操作）
3. 对流中数据的整合：转为集合、数量。。（最终操作）
