## 更新安装命令



sudo apt update
sudo apt install unrar


安装完成后，可以使用以下命令解压当前目录下的 .rar 文件到当前目录：

unrar x filename.rar
参数解析：
unrar: 这是解压缩工具的命令。
x: 这个参数表示提取文件到当前目录，并保持目录结构。
filename.rar: 这是你要解压的 .rar 文件的名称。
如果你想查看 unrar 的其他可用选项，可以使用以下命令：

unrar
