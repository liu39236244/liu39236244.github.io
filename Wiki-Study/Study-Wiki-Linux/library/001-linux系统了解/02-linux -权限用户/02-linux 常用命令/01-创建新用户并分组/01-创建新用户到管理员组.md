好的，以下是在 Ubuntu 系统上以 root 或 sudo 权限为你完成这些操作的详细命令。
假设你当前账户有 sudo 权限。

---

### 1. 创建新用户 yabo

```bash
sudo adduser yabo
```

系统会提示你输入密码，请输入 `yabo123`，并根据需要填写其他信息（也可以直接回车跳过）。

---

### 2. 将 yabo 用户加入管理员组

在 Ubuntu 18.04 及以上版本，管理员组一般是 `sudo`。在老旧系统可能是 `admin`，但主流新系统基本都是 `sudo`：

```bash
sudo usermod -aG sudo yabo
```

这样 `yabo` 就具有 sudo 权限，可以执行管理员操作了。

---

### 3. 检查

你可以这样验证：

```bash
groups yabo
```

返回类似如下内容，说明成功：

```bash
yabo : yabo sudo
```

---

**注意：**

- 有些情况下可能需要重新登录/重启才能生效。
- 直接用命令设置明文密码（不安全，不建议，但可行）：

  ```bash
  echo 'yabo:yabo123' | sudo chpasswd
  ```

---

### 总结脚本

如果你想一条命令跑下来，可以用：

```bash
sudo adduser --disabled-password --gecos "" yabo
echo 'yabo:yabo123' | sudo chpasswd
sudo usermod -aG sudo yabo
```

这样即可快速完成所有操作。
